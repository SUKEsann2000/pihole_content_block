# Description

Pihole_blocking on Blowser

Pi-holeにおいてブロックされ、エラーを吐いてそのままのエレメントを削除します。

普通に404吐いてるiframeとかも削除しますので注意してください。

## What do this program do?

1. DOMのロードまで待つ
2. エラーを起こしているエレメントを探す
3. そのエレメントを0.5秒後に削除
